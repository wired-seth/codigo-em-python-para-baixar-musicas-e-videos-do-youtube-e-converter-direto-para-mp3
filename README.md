Instale o yt-dlp e o ffmpeg (necessário para converter em MP3)

comando para instalar o ty-dlp (execute no cmd)

  pip install yt-dlp

link para o ffmpeg

https://www.gyan.dev/ffmpeg/builds/#release-builds

Salve o codigo como baixar_musica.py na pasta "C:\Users\User"

para usar basta executar o comando no cmd:

python baixar_musica.py


Cole o link da playlist ou musica e espere baixar.
